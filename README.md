## Cargo

Criando projeto

```cargo new hello_world --bin```

Compilando e baixando crates

```cargo build```

Executando 

```cargo run```

Verificação

```cargo check```

### Gerando versão de release

```cargo build --release```

### Atualizando crates

```cargo update```

### Gerando documentação 

```cargo doc --open```
